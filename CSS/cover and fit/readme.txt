use to maintain a image in a specific position

object-fit:cover; - cut the image to fully cover the parent area
object-fit:contain; - cut the image to maintain he image aspect ratio with no cut to the image, it may lead to extra space in conatiner
object-fit:fill; - fill the image


to decide which portion you want to cut: use object-postion : to right/ bottom rigth etc
object-position: 10px 20px; - describe image position wrt cover or contain

background-img: to set background of a image
if the image is small it by default repats
to stop repeat use
    background-repeat: no repeat;
    background-repeat: repeat-x;
    background-repeat: repeat-y;
